# 🛒 E-Commerce Website

A full-stack E-Commerce web application where users can browse products, add them to cart, and place orders online.

---

## 🚀 Features

- User Registration & Login
- Product Listing
- Product Details Page
- Add to Cart
- Order Placement
- Admin Product Management
- Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB
